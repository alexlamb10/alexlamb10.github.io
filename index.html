Work

Sell Monkey

This application was my final capstone project for the Devmountain Bootcamp. This project was created using Node JS, Postgres, Express, React as well as Auth0 for authentication, AWS S3 to store pictures, and Stripe to handle payments. The application enables users to buy and sell items. Items that are listed for sale are saved in a database on heroku, along with the user Id that auth0 provided. Users are able to view items other people have listed and add them to their cart before purchasing. Users are then able to use their Credit Cards to purchase items. Stripe is currently in a testing state, this means only Stripe testing cards will work.

A demo of this project can be found here. This app has also been deployed on Heroku and the code can be found on GitHub.

Trip Planner Application

I made this application for my first capstone project while I was attending Devmountain. I used HTML, CSS, Javascript, Axios, Cors, Postgres, Postman, and AWS S3 in the app. This app allows users to create an account, in which their password is stored as a password hash. The user can then log in using their account credentials and begin adding trips to a wishlist of places they may want to travel to. Users then can navigate to a page that displays the trips they have added to their wishlist. On this page the user can either delete trips or mark the trip as complete. If it is marked as complete it no longer appears on this page. The last page, shown above, is the page that displays the users completed trips of the user. On this page the user can also add a picture from their trip. The pictures are stored using AWS S3, which returns a URL. The URL, as well as the rest of the information received from the user, is stored in tables on the database.

A demo of this project can be found here. This app has been deployed on Heroku and the code can be found on GitHub.
